# FAQ

Frequently asked questions about KillerTools.

## General

### What is KillerTools?
KillerTools is a modern, cross-platform Swiss-army toolkit for developers and makers with plugin architecture and multiple interfaces.

### What platforms are supported?
Windows, macOS, and Linux.

### How do I install KillerTools?
See [Installation Guide](installation.md).

## Plugins

### How do I create a plugin?
See [Creating Plugins](creating-plugins.md).

### Can I install third-party plugins?
Plugin marketplace coming soon. For now, plugins must be in the `killer_tools/plugins/` directory.

### How do plugins work?
Plugins are auto-discovered and registered. They implement the Plugin protocol.

## Usage

### How do I change the theme?
Use the theme setting in configuration or `--theme` option.

### Can I use multiple interfaces?
Yes, you can use CLI, TUI, and GUI independently.

### How do I configure settings?
Edit `~/.killertools/config.json` or use environment variables.
